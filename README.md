# Think_Cricket
Cricket App - Let's go 










## SQL Queries
### Player
Id will be generated by application.

    CREATE TABLE player (
        id VARCHAR(50) PRIMARY KEY,
        name VARCHAR(255),
        age INT,
        gender ENUM('Male', 'Female')
    );


### Ball  

    CREATE TABLE ball (
        id VARCHAR(50) PRIMARY KEY,
        run INT,
        batsman_id VARCHAR(50),
        non_striker_id VARCHAR(50),
        legal_delivery BOOLEAN,
        over_id VARCHAR(50),
        ball_number INT
    );
